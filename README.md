**Multi-Ligand Docking and Molecular Dynamics using Linux**

A Linux-based workflow for performing multi-ligand molecular docking and molecular dynamics (MD) simulations for in-silico drug discovery.

This repository provides command-line pipelines, scripts, and step-by-step procedures to automate docking and simulation tasks efficiently.

## Project Overview
This project presents a Linux-based workflow for performing multi-ligand molecular docking and molecular dynamics (MD) simulations for in silico drug discovery. It demonstrates the use of command-line tools and scripting to automate and streamline computational analysis.

## Objective
The aim of this project is to identify potential therapeutic compounds by analyzing protein–ligand interactions and evaluating the stability of complexes using molecular docking and molecular dynamics simulations.

## Key Concepts
- **Molecular Docking:** Predicts ligand binding affinity with target proteins  
- **Molecular Dynamics (MD):** Studies the stability and behavior of protein–ligand complexes over time  

## Features
- Multi-ligand docking workflow  
- Automation using Bash scripting  
- Linux command-line based pipeline  
- Molecular dynamics simulation setup  
- Reproducible and organized workflow  

## Tools Used
- AutoDock Vina – molecular docking  
- Open Babel – file format conversion  
- GROMACS – molecular dynamics simulation  
- MGLTools – receptor and ligand preparation  
- Bash scripting – workflow automation

  ## Workflow
1. Protein preparation (target receptors such as EGFR, HER-2)  
2. Ligand preparation and optimization  
3. Molecular docking using AutoDock Vina  
4. Analysis of binding energies and interactions  
5. Molecular dynamics simulation using GROMACS  
6. Visualization and interpretation of results

## System Requirements
- Linux OS (Ubuntu recommended)  
- Python (3.x)  
- Installed tools:
  - AutoDock Vina  
  - Open Babel  
  - GROMACS  
  - MGLTools
